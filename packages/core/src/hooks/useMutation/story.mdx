# useMutation

Basic mutation hook ( it's not expected to use directly, only combine with other high level, entity hooks)


```js
export const useSomeEntityAdd = () => {
    const { mutate, isLoading, error, data } = useMutation(mutateFn, options)
    return { isLoading, error, add: mutate }
}
```

### Options
* `mutateFn` - plain fetch function to launch when mutate method is called

### Returns

* `data: unknown` - Response data.
* `mutate: () => void` - Triggers action call.
* `isLoading: boolean` - Reports loading state.
* `error: null | any` - Reports error state.
