# useMapsEntity

```js
const { data, loading, refetch, fetchMore, error } = useMapsEntity({
    initialData: useMapsEntity.initialData
})
```

### Options

The options for `useMapsEntity` are identical to the `useRequest` [hook](/?path=/docs/hooks-useRequest--page).

### Returns

The returned properties for `useMapsEntity` are identical to the `useRequest` [hook](/?path=/docs/hooks-useRequest--page).

### Static

* **.initialData()** - default initial data function. This data will be shown at the loading stage.
